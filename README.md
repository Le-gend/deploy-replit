## Install
Clone this project

```bash
> https://github.com/inirey/whatsapp-bot.git
> cd whatsapp-bot
```

Install the dependencies:

```bash
> bash install.sh
```

### Usage
Run the Whatsapp bot

```bash
> npm start
```
